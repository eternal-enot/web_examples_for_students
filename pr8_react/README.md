# Запуск `pr8_react` (React)

```bash
# 1. Перейти в папку pr8_react
cd pr8_react

# 2. Встановити залежності (виконується один раз)
npm install

# 3. Запустити dev-сервер
npm run dev
```

Після запуску відкрийте адресу, яку покаже Vite (зазвичай `http://localhost:5173/`).

## Що реалізовано

- React-версія сайту з `pr3_simple`
- Дві сторінки через роутинг (`react-router-dom`): головна та контакти
- Перемикання мови `UA/EN` (через React Context)
- Перемикання теми з `localStorage` (через React Context)
- Фільтр проєктів за тегами
- Модальне вікно деталей проєкту
- Кнопка `Back to top`
