# Запуск `pr8_angular` (Angular)

```bash
# 1. Перейти в папку pr8_angular
cd pr8_angular

# 2. Встановити залежності (виконується один раз)
npm install

# 3. Запустити dev-сервер
npm start
```

Після запуску відкрийте адресу, яку покаже Angular CLI (зазвичай `http://localhost:4200/`).

## Що реалізовано

- Angular-версія сайту з `pr3_simple`
- Дві сторінки через роутинг: головна та контакти
- Перемикання мови `UA/EN`
- Перемикання теми з `localStorage`
- Фільтр проєктів за тегами
- Модальне вікно деталей проєкту
- Кнопка `Back to top`
