# Запуск `pr8_vue` (Vue)

```bash
# 1. Перейти в папку pr8_vue
cd pr8_vue

# 2. Встановити залежності (виконується один раз)
npm install

# 3. Запустити dev-сервер
npm run dev
```

Після запуску відкрийте адресу, яку покаже Vite (зазвичай `http://localhost:5173/` або `http://localhost:5174/`).

## Що реалізовано

- Vue-версія сайту з `pr3_simple`
- Дві сторінки через роутинг (`vue-router`): головна та контакти
- Перемикання мови `UA/EN` (через Vue Composables)
- Перемикання теми з `localStorage` (через Vue Composables)
- Фільтр проєктів за тегами
- Модальне вікно деталей проєкту
- Кнопка `Back to top`
