# Запуск `pr6`

```bash
# 1. Перейти в корінь репозиторію
cd "/шлях/до/Web"

# Альтернатива для Windows:
# cd C:\шлях\до\Web

# Альтернатива для Linux:
# cd /шлях/до/Web

# 2. Встановити залежності
npm install

# 3. Перевірити TypeScript-код для pr6
npm run check:pr6

# 4. Зібрати pr6
npm run build:pr6

# 5. Відкрити папку проєкту у VS Code
# 6. Встановити розширення Live Server
# 7. Відкрити файл pr6/index.html
# 8. Натиснути Go Live

```
